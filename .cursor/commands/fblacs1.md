This project must strictly follow the official FBLA brand color scheme.

Primary Colors:

Supporting Colors:
- Cobalt Blue: #226ADD (icons, subtle accents)
- White: #FFFFFF (main backgrounds, cards)
- Black/Dark Gray: #2D2B2B (primary text)

Rules:
- Do NOT introduce random or trendy colors.
- Gold (#F4AB19) is used sparingly for emphasis and calls-to-action only.
- Maintain high contrast for accessibility (WCAG AA minimum).
- Default UI style should be clean, modern, and professional.
- If unsure, prefer Navy + White with Gold accents.

All UI components, themes, and generated styles must comply with this palette.
